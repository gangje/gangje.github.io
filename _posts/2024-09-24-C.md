1) 매개변수, 반환형이 둘 다 없는 함수
   예) int get_num(int a, int b)

2) 매개변수가 없고, 반환형은 있는 함수
   예) int get_num(void)

3) 매개변수가 있고, 반환형이 없는 함수
   예) void get_num(int a, int b)

3) 매개변수, 반환형 둘 다 없는 함수
   예) void get_num(void)

4)재귀 함수 :함수안에 본인 함수가 포함되어있다
  예) void recursive() {... recursive();...}

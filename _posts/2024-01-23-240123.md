챕터 3-1 회귀,과대적합/과소적합<br>
===
■회귀: 분류가 새로운 데이터의 클래스를 분류하는것이라면 회귀는 새로운 데이터의 예측에 사용, 분류와의 공통점은 훈련데이터와 테스트 데이터로 나누어 훈련한후 모델의 적합성을 테스트 하는것에 있음 ,대표적으로 k-최근접 이웃 회귀가 있음<br>

■결정계수(R^2): 회귀 모델의 성능 측성 도구 ,1에 가까울 수록 예측에 좋은 모델이며 0에 가까울수록 예측에 나쁜 모델이다<br>

R^2=1- 시그마(타깃-예측)^2÷시그마(타깃-평균)^2의 식으로 계산된다<br>

■k-최근접 이웃 회귀: k-최근접 이웃 분류와 비슷하게 n개의 근처값의 평균값으로 데이터를 예측 차이점은 k-최근접 이웃 분류는 근처 데이터의 클래스들의 다수결로 분류모델을 만들었다면, k-최근접 이웃 회귀는 근처 데이터들의 임의로 부여된 수치들의 평균값을 이용해 예측모델을 훈련한다는 차이가 있음, 이웃의 디폴트 값(n)은 5, 즉 주위 5개의 값을 사용하며, 고려할 이웃데이터(n)의 갯수가 많아질수록 훈련모델은 단순화(전면적)가 되며, 이웃 데이터(n)의 갯수가 작아질수록 복잡화(국지적)이 된다<br>

■과대적합:회귀 예측 모델이 테스트데이터에서보다 훈련데이터에서 더 좋은 예측값을 나타내는 현상, 이는 테스트 데이터와 새로운 데이터에 대한 올바른 예측을 하지 못함으로 문제가 될수 있다, 즉 훈련데이터 예측의 편향
해결법) 이웃 데이터(n)의 개수의 갯수를 늘려 전반적인 데이터의 흐름을 반영하도록, 즉 단순화(전면화) 함으로 해결한다<br>

■과소적합:과대적합과 반대되는 현상. 회측 회귀 모델이 훈련데이터에서 보다 테스트데이터에서 예측값이 더 높은 현상 또는 훈련데이터와 테스트 데이터 둘다 예측값이낮은 현상, 이는 훈련데이터의 값들을 제대로 훈련하지 못했다는것을 의미함 
해결법)모델 훈련때 훈련 데이터들의 값들을 더욱 민감하게 반영할수 있도록 k-최근접 이웃 회귀에 사용되는 근처 이웃 데이터의 값(n)을 줄임으로 즉, 단순화(국지화) 함으로 해결 한다<br>

챕터 3-2 선형회귀,다항회귀
===
■선형회귀: k-최근접 이웃회귀는 훈련세트에서 벗어난 데이터가 입력될시 그 근처의 값들의 평균으로만 데이터를 나타내기 때문에 실제로 멀리 떨어져있는 데이터의 경우일지라도 최근처의 값이기에 평균치에 반영되어 올바른 모델 학습이 되지 않는 문제점이 있음 이를 해결하고자 선형회귀를 사용, 선형회귀는 일차함수, 즉 직선의 함수 ax+b=0 에서 기울기(a)와(b)의 최적값들을 모델학습을 통해 알아낸다<br>

■모델파라미터: 회귀 알고리즘이 찾은 최적값들을 말함, 위의 선형회귀 같은경우는 기울기(a)와 절편(b)를 말함<br>

■다항회귀: 선형회귀의 최적값은 결국 직선의형태이기에 일반적인 경향에서 벗어나게 되는 문제가 있음 이를 해결하고자 직선을 곡선으로 즉, 일차함수에서 -> n차 함수로의 곡선의 모델 파라미터를 구하고자 하는것 
일반적으로 이차방정식 그래프의 경우 ax^2+bx+c=y 에서 a,b,c의 모델 파라미터를 구하는것이 다항회귀 모델의 목적이 된다.<br>
주의사항) 다항회귀를 선형회귀라고 부를 수 있는가? =>가능하다, 위의 이차방정식 그래프에서 x^2이 있고, 학습모델이 비선형이기에 선형회귀가 아닐것 같지만 x^2을 하나의 단일 덩어리변수로 본다면 이것역시 선형회귀라고 부를 수 있디.<br>
